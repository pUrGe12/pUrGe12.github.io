pyba

This is a no-code browser automation framework library for python. I am developing this because it is a major step for my main project (Atlas) and there are many beautiful things that PyBA can do. For starters, all you need to do is enter a simple prompt, and it will visit the relevant sites, login to social media if needed (without passing your credentials to the LLM), create its own DSL (domain specific language), provide tracibility and database options and reproduce the entire automation scrpit in playwright form scratch!
